# Glossary

| Term | Meaning |
| --- | --- |
| Agent | A specialist reviewer with its own prompt and default model. |
| Aggregator | Merges, dedupes, ranks, and renders findings into one PR comment. |
| Chunk | A merged group of nearby hunks from a single file. |
| Finding | One reviewer comment, with severity, category, file, line. |
| Pipeline | The full review path: parse, chunk, dispatch, aggregate. |
| Installation | A GitHub account that installed the App. |
