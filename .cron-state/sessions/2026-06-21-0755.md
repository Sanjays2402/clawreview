## What landed

- **Worker rewire (#1):** The post-suppression bucket recompute was 12 lines
  of bespoke `for f of findings` arithmetic. Replaced with `recomputeAggregateTotals(result)`
  which delegates to `findingDigest()` — the same helper the CLI's `stats`
  command consumes. Worker / CLI / dashboard now share the same arithmetic
  end-to-end; a future bucket added to `findingDigest` automatically surfaces
  in all three without manual edits.

- **Comment header rewire (#2):** Added `topAgents` / `topCategories` / `digest`
  opts to `renderPrComment`. When set, the category line consumes the digest's
  pre-capped slice (with a `_(N more)_` tail when truncated). New `By agent:`
  header line, off by default. Worker now passes 8/8 caps so the PR comment
  matches the CLI's `stats --by agent --top-agents 8` output byte-for-byte.

- **Operator-poll counter (#3):** `clawreview_operator_poll_total{probe,result}`
  bumps once per polling request on `ok` / `bypass` / `throttled`. Probe label
  inherits the tick-10 `?probe=name` annotation; `(none)` for anonymous polling.
  Operators can now graph `rate(...{result="throttled"}) by (probe)` to spot
  the noisiest widget without grepping logs.

- **Presets diff filter (#4):** `--only-fields severity_threshold,min_confidence`
  scopes the diff to a focused subset of top-level keys. CI-gateable exit-3 only
  fires on in-scope drift; a delta hidden by the filter exits 0. Text / yaml /
  json renderers all surface the active filter; text gets a distinct
  "(no differences in the filtered fields)" hint when the filter excluded
  everything. `parsePresetOnlyFields` exposes an `ONLY_FIELDS_EMPTY_ENTRY`
  sentinel so the caller can refuse a stray-comma input rather than silently
  widening.

- **Dotted-path payloadFields (#5):** Widens `projectPayload` so each entry in
  the allowlist may be a top-level key OR a dotted path. `pull_request.title`
  walks the source tree and writes the value into a mirrored nested output
  (`{ pull_request: { title } }`), so dashboards consuming GitHub's payload
  shape don't have to learn a flat DSL. Multiple paths under one prefix merge
  cleanly. Depth capped at 6; array index syntax NOT supported (callers can
  fetch full entry via `get()`). Tick-10 top-level callers see byte-identical
  output — back-compat is a test pin.
